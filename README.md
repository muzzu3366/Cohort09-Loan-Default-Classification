# Cohort09-Loan-Default-Classification
Machine Learning project developed by Cohort 09 to predict loan default risk using financial and borrower data.
## Project Overview
This project focuses on predicting whether a borrower is likely to default on a loan using machine learning techniques. Financial institutions face significant challenges in identifying high-risk borrowers before approving loans. By analyzing historical borrower and financial data, machine learning models can help make better lending decisions and reduce financial risk.

This project is developed as part of the **Cohort 09 Machine Learning Internship Program**, where our team applied data preprocessing, model building, and model evaluation techniques to build a loan default prediction system.

## Problem Statement
Loan defaults can cause major financial losses for lending institutions. The goal of this project is to develop a machine learning model that can analyze borrower information and predict whether a loan applicant is likely to default.

## Project Objectives
- Analyze borrower financial data
- Build a machine learning classification model
- Predict loan default risk
- Evaluate model performance using multiple metrics
- Identify key factors influencing loan default

## Dataset
The dataset used in this project contains borrower related information such as:

- Credit Score
- Monthly Income
- Debt-to-Income Ratio
- Loan Amount
- Interest Rate
- Employment Years
- Repayment History
- Dependents

To improve experimentation and model training, the dataset was also extended using **synthetic data generation techniques** to simulate realistic financial scenarios.

## Machine Learning Model
The primary model used in this project is:

**Random Forest Classifier**

This model was chosen because it handles classification problems effectively and works well with structured financial datasets.

## Model Evaluation Metrics
The model performance is evaluated using multiple metrics:

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1 Score
- ROC Curve
- AUC Score
- Gini Coefficient
- Precision-Recall Curve
- Feature Importance Analysis

## Results
The trained model achieved approximately **74% accuracy**, demonstrating its ability to identify loan default risk.

Feature importance analysis showed that the most influential factors include:

- Credit Score
- Debt-to-Income Ratio
- Loan Amount
- Monthly Income
